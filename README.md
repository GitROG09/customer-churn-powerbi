## Project Overview
This project focuses on analyzing customer churn using historical customer data. 
The objective is to identify churn patterns, key drivers, and high-risk customer segments 
using interactive Power BI dashboards.

## Tools Used
- Microsoft Power BI
- Power Query
- DAX

## Key Analysis
- Churn Rate Analysis
- Tenure vs Churn Trend
- Contract Type vs Churn
- Payment Method vs Churn
- Internet Service vs Churn

## Dashboard Features
- KPI Cards for churn metrics
- Interactive filters
- Insight and recommendation sections

## Outcome
The dashboard provides actionable insights to help businesses understand why customers churn 
and supports data-driven retention strategies.

## Note
This project focuses on descriptive and diagnostic analytics, not churn prediction.
